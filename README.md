dinglehopper
============

dinglehopper is an OCR evaluation tool and reads ALTO, PAGE and text files.

![Travis CI badge](https://api.travis-ci.org/qurator-spk/dinglehopper.svg?branch=master)

Goals
-----
* Useful
  * As an UI tool
  * For an automated evaluation
  * As a library
* Unicode support
